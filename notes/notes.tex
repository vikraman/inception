\begin{definition}[Monad]
    \label{def:monad}
    A monad $T$ on a category $\cat{C}$ is an extension system $(T, \eta, (\blank)^*)$.
\end{definition}

\begin{theorem}[Kock]
    \label{thm:kock}
    If $S$ is a strong monad on a ccc $\cat{C}$, then $S$-algebra structures on $V$ are in bijection with monad morphisms $S \To K_V$.
\end{theorem}

\begin{proof}
    \leavevmode
    Define $m_X$ by currying $SX \times V^X \xto{1 \times (\blank)^\#} SX \times V^{SX} \xto{\ev} V$.
    Given $f\colon X \to V$, define $f^\# \defeq m_X(f)$.
\end{proof}

\begin{proposition}
    \label{prop:algebra-along-morphism}
    A monad morphism $m\colon S \To T$ induces a functor $T$-Alg $\to$ $S$-Alg.
    In particular, this means $TX$ is an $S$-algebra for any $X$.
\end{proposition}

\begin{proof}
    Given $f\colon Y \to X$, define $f^\# \defeq STX \xto{m_{TX}} TTX \xto{\mu_X} TX $.
\end{proof}

In our model, $S$-algebra structure on $V$ induces a monad morphism $S \To K_V$, which induces an $S$-algebra structure
on $K_V(X)$ for any $X$. Now, we use this to get the maps $\var$ and $\sub$ for $K_V(X)$, which satisfy the substitution
algebra axioms.
